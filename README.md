> > How to make footer stay at the bottom of the page?

```
  <body>
    <div class="container">
      <div class="header">...</div>
      <div class="main">...</div>
      <div class="footer">...</div>
    </div>
  </body>
```

- set css to `div.container`:

  - `min-height: 100vh`
  - `display: flex`
  - `flex-direction: column`

- set css to `div.main`
  - `flex-grow: 1`
